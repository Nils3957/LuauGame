# LuauGame (Roblox Project)

This project is a Roblox game development repository using Rojo.

## 🚀 Key Features
*   **Double Jump:** Double jump capability for characters (StarterCharacterScripts)
*   **Shift to Run:** Sprinting functionality when holding the Shift key (StarterCharacterScripts)
*   **Rojo Integration:** Configuration for syncing external code editors with Roblox Studio

## 🛠 How to Use
1.  Clone this repository.
2.  Run `rojo.exe` to open the server:
    ```bash
    ./rojo.exe serve
    ```
3.  Connect using the Rojo plugin in Roblox Studio.

## 📂 Project Structure
*   `src/client`: Client-side scripts (UI, character control, etc.)
*   `src/server`: Server-side scripts (Game logic, database, etc.)
*   `src/shared`: Modules shared between server and client

*   **Latest Update:** Optimized DoubleJump functionality and adjusted sounds

## 🇰🇷 한국어 설명 (Korean Description)

이 프로젝트는 Rojo를 활용한 로블록스 게임 개발 저장소입니다.

### 🚀 주요 기능
*   **Double Jump:** 캐릭터의 2단 점프 기능 (StarterCharacterScripts)
*   **Shift to Run:** Shift 키를 눌러 달리기 기능 (StarterCharacterScripts)
*   **Rojo Integration:** 외부 코드 에디터와 로블록스 스튜디오 동기화 설정

### 🛠 사용 방법
1.  이 저장소를 클론합니다.
2.  `rojo.exe`를 실행하여 서버를 엽니다:
    ```bash
    ./rojo.exe serve
    ```
3.  로블록스 스튜디오에서 Rojo 플러그인을 사용하여 연결합니다.

### 📂 프로젝트 구조
*   `src/client`: 클라이언트 사이드 스크립트 (UI, 캐릭터 제어 등)
